03The morning run — five workers, one pipeline

One button (or the 9:00 weekday scheduler). Everything below is deterministic code — the only LLM calls in the entire engine are reply classification and the tailored reply draft.

Scout

research

Takes the brief: niche + locations + lead target. You pick niche from a dropdown; each location becomes its own search. "med spas in Union NJ | med spas in Elizabeth NJ" = one Apify run, two searches.

Harvester

scraper

Bank first. Yesterday's un-emailed leads are free inventory — if they cover today's target, $0 spent. Only scrapes when short, max 2 runs per day, over-fetching 3× the deficit because verification kills a chunk.

Inspector

email finder

Per lead: AnyMailFinder decision-maker ("CEO") search → if none indexed (typical for local spas), falls back to any validated email at the domain, preferring personal addresses over info@. Valid only — risky/unknown are skipped forever (bounces are what kill inboxes). Dead leads are remembered, never re-checked.

Courier

sender

Starts sending the moment the first verified lead exists — runs concurrently with scraping/verification. Rotates inboxes under per-inbox daily caps, 9–14s random gaps, business hours only (Mon–Fri 9–5). Schedules exactly one follow-up (+3 days). Leftovers → bank.

Concierge

replies · 24/7

Polls all inboxes every 2 minutes. Any reply instantly kills follow-ups for that lead, then: interested/neutral → short lowercase answer + patient-engine link, same thread · negative → Telegram flag, no link, human decides · unsubscribe → do-not-contact forever, no reply, no exceptions.

04Sending safety — the rules that protect the inboxes

Verified emails only. A lead without a validated address never gets sent to. This is the #1 rule; bounces are what get domains flagged.
Per-inbox daily caps (default 20, new Zapmail inboxes start at 10), volume spread across all inboxes — adding inboxes thins the load automatically.
Random 9–14 second gaps between sends, business hours only, exactly one follow-up per lead, ever.
Test mode until you flip it: every email diverts to your own Gmail with the real lead's address in the subject. Live mode is an explicit switch.
Warmup runs forever (external tool / Zapmail) on every inbox in the pool — pre-warmed ≠ warm forever.
