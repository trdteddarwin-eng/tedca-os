When a lead writes back

Reply handling

Reading replies runs free through the Gmail API — no $96 Instantly tier. Any reply stops follow-ups. What happens next depends on the message:

Interested

Agent replies with a tailored message + your patient-engine link — “a system already built for you.” Marks Replied, stops follow-ups, Telegrams you.

Negative

Not interested, but not unsubscribe. Agent flags it + Telegrams you, sends no link, stops follow-ups. You decide.

Unsubscribe

“Stop emailing me.” Hard stop — suppressed, marked Do-Not-Contact, never emailed again. No reply, no follow-up. No exceptions.

Trust the agent, but verify

In the loop, self-healing, on budget

Three guarantees that make “let the agent run it” safe.

Telegram HITL

It waits for you

When an agent needs a decision, it Telegrams you and freezes — polling until you reply — then resumes exactly where it paused. Nothing moves without you.

Self-healing

It fixes itself

On failure: capture the error → OpenRouter diagnoses → retry with a fix. Still failing → stop, Telegram the reason, log to error-log.md. No silent failures.

Cost tracking

It counts the money

Every call logged with cost → per-run + a running monthly total on Home. Anything over $0.50 asks first.
